# Privacy Policy for YouTube Rec Sort

_Last updated: July 14, 2026_

YouTube Rec Sort is a browser extension that sorts recommendations on the YouTube Home page by view count and can hide videos older than two years, Shorts, advertisements, and other non-video sections.

## Data handled by the extension

The extension reads content already displayed on the YouTube Home page, including recommendation cards, view counts, publication age text, and links. This information is processed locally in the user's browser only to provide the extension's sorting and filtering features.

The extension also stores two preference settings: whether sorting by view count is enabled and whether the two-year age filter is enabled. These settings are stored through the Chrome Storage API. When Chrome Sync is available and enabled, Chrome may synchronize these settings between the user's devices according to the user's browser settings.

## Data collection and transmission

YouTube Rec Sort does not transmit YouTube page content, browsing activity, personal information, or extension usage data to the developer or to any third-party server.

The extension does not use analytics, advertising trackers, cookies, remote code, or external APIs.

## Data sharing and sale

The developer does not sell, rent, share, or disclose user data to third parties.

## Data retention

YouTube page content is processed temporarily in the browser and is not retained by the extension. The saved preference settings remain in Chrome Storage until the user changes them, clears extension data, or uninstalls the extension.

## Limited use

Any data accessed by the extension is used only to provide or improve its stated user-facing sorting and filtering features. It is not used for advertising, profiling, creditworthiness, or any unrelated purpose. No human has access to the data processed by the extension.

## Changes to this policy

If the extension's data practices change, this privacy policy will be updated before or together with the relevant extension update.
